### Project Setup

Clone the repository using

```shell
git clone https://github.com/erandakarachchi/algodashboard.git
```

Install dependencies

```shell
npm install
```

Run the project

```shell
npm start
```

**Algo dashboard** will be running on `http://localhost:3000/`
